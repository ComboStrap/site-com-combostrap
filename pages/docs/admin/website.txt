====== ComboStrap WebSite ======

===== About =====

A Combostrap WebSite is a collection of files that represents a [[:docs:app:app|ComboStrap WebSite App]].

It permits to develop and publish website collaboratively (locally or remotely) using Git.
===== Example =====

==== Default WebSite ====

The [[https://github.com/ComboStrap/site-default|Default WebSite]] is a website that is installed by default with [[:docs:admin:docker|DokuWiki Docker]]
when no site is specified.


===== Layout =====

A ComboStrap WebSite is a git repository with the following files and directory:
  * ''.gitignore'' - defines the files of a DokuWiki Installation that should be committed in git.
  * ''data/pages/*'' - all [[:docs:page:system:system|pages]]
  * ''data/media/*'' - all media (images, files, ...) 
  * ''data/combo/theme'' - the combo theme ([[:howto:theme-creation|theme]], ...)
  * ''data/combo/instance'' - the [[#plugins|plugins definition]]
  * ''conf/local.php'' - the [[doku>config|configuration]]
  * ''conf/*.local.*'' - all local configuration
  * ''lib/images/interwiki'' - the [[doku>interwiki|interwiki images]]


==== Plugins ====

The plugins definition file is used by the [[docs:admin:docker|ComboStrap DokuWiki Docker Image]] to install the needed plugins of your app.

Example from the [[https://github.com/ComboStrap/site-default/tree/main/data/combo/plugins/plugins.txt|default ComboStrap Website]]

<file data/combo/instance/plugins.txt>
# The plugins installed by DokuWiki Docker (https://combostrap.com/admin/docker-9iq3aso8)
sqlite       https://github.com/cosmocode/sqlite/archive/main.zip
combo        https://github.com/ComboStrap/combo/archive/main.zip
move         https://github.com/michitux/dokuwiki-plugin-move/archive/main.zip
searchindex  https://github.com/splitbrain/dokuwiki-plugin-searchindex/archive/main.zip
stale        https://github.com/ComboStrap/stale/archive/main.zip
dump         https://github.com/ComboStrap/dump/archive/main.zip
</file>



The syntax of the file is:
<file data/combo/instance/plugins.txt>
pluginName  pluginArchiveUrl 
</file>
where:
  * ''pluginName'' is the plugin name (ie the ''base'' property defined in the [[doku>devel:plugin_info|plugin_info]] file)
  * ''pluginArchiveUrl'' is the URL of a zip archive. Example:
      * The last commit on the main branch: https://github.com/ComboStrap/combo/archive/main.zip
      * The release [[:release:1.25.3|1.25.3]] from the [[https://github.com/ComboStrap/combo/releases/tag/v1.25.3|GitHub Release]] by:
        * its tag name: https://github.com/ComboStrap/combo/archive/refs/tags/v1.25.3.zip
        * its commit hash: https://github.com/ComboStrap/combo/archive/ec02c60.zip
