====== Step by step: How to set a configuration value======

===== About =====
This page shows you how to set a configuration value that is valid for your [[app|whole application]]


<note tip>
The configurations that are only valid for a page are called [[docs:metadata:metadata|metadata]]
and can be changed via the [[docs:metadata:manager|metadata manager]].
</note>

===== Steps =====

==== Go to the Admin Interface ====

To set a configuration value, go to the [[doku>admin_window|Admin Window]]

  * With the [[:strap|Strap template]]

{{::dokuwiki_admin_manager.png|}}

  * With the default template

{{::dokuwiki_admin_window_default_template.png|}}

==== Go to the Configuration Manager ====

  * With the [[:strap|Strap template]]

{{::dokuwiki_configuration_settings_Strap.png|}}

  * Or with the default template

{{::dokuwiki_configuration_settings_default_template.png|}}


==== Values ====

  * Change the configuration
  * And hit the ''save'' button

<note tip>
  * A blue background shows default values (ie values that have not been modified)
  * A white background indicates local changes. (ie values that have been modified)
</note>


===== Documentation / Reference =====
This configuration interface is the [[doku>plugin:config|Dokuwiki Configuration Manager Plugin]].
