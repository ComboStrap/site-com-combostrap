====== ComboStrap UI - Header ======


===== About =====
''header'' is the top location of a page or of a component.

Combostrap supports:
  * the modification of the page header with the [[docs:layout:page:header_slot|header slot]]
  * the definition of an header in the following component:
      * [[docs:block:card]]
      * [[docs:block:blockquote]]
