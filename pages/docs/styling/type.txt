====== ComboStrap Styling - Type Attribute ======


===== About =====
''type'' is the first [[styling|attribute]] of [[..:marki|the combostrap marki syntax]].

The default behavior of a ''type'' is to change the style.

<note>
If you want to apply custom styling, you can also create you own rules using the [[docs:styling:userstyle#custom|combo styling class]] that contains the ''type''.
</note>




===== Example =====


Example with a [[docs:block:note|note]] that has the ''info'' type.
<code dw>
<note info>A info note</note>
</code>
