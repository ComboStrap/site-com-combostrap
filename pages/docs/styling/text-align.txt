====== ComboStrap Styling - Text Align Attribute ======


===== About =====
''text-align'' is a [[docs:content:typo|typographic attribute]] that sets the alignment of the lines of text inside a [[docs:block:block|block component]].

===== Syntax =====

<code dw>
align=text-value[-breakpoint]
</code>
where:
  * ''breakpoint'' is optional and takes one of the [[docs:styling:breakpoint|breakpoint values]] (ie screen size) 
      * ''sm'': ''small''
      * ''md'': ''medium''
      * ''lg'': ''large''
      * ''xl'': ''extra-large''
  * the value may be:
    * ''start'' - left for a LTR language and right for a RTL language - (default)
    * ''center''
    * ''end'' - right left for a LTR language and left for a RTL language


<note>
''justify'' is not supported. While, aesthetically the text might look more appealing, it does make word-spacing more random and therefore harder to read.
</note>
===== Demo =====

==== Basic ====
This example uses the [[docs:block:text|text component]] (to add a margin space between the lines) but it should work for all other [[docs:block:block|block components]].


<webcode name="Default" frameborder="0">
<code dw>
<text align="text-start">
This line of text is at the **start** of the block (left for a left-to-right language)
</text>
<text align="text-center">
This line of text is **centered** inside the block.
</text>
<text align="text-end">
This line of text is at the **end** of the block (right for a left-to-right language)
</text>
</code>
  * Output:
</webcode>
==== Centered text in a card ====

A centered text in a [[docs:block:card|card]]:
<webcode name="Text Align" >
<code dw>
<card text-align="center" width="300px">
==== Centered Text in a Card  ====

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

</card>
</code>
</webcode>
==== Screen Conditional ====

This example was made with the [[docs:block:text|text component]] and shows a conditional value based on [[docs:styling:breakpoint|screen size]]


<webcode name="Default" frameborder="0">

  * Input
<code dw>
<text align="text-end-sm">End aligned text from SM (small) screen and wider.</text>
<text align="text-center-md">Default (Start) align on SM (small screen), then Center aligned from MD (medium) screen and wider.</text>
<text align="text-center-md text-end-lg">Default (Start) align on SM (small screen), then Center aligned from MD (medium), then End aligned text on LG (large) screen and wider.</text>
<text align="text-center-xl">Default (Start) align on SM (small screen), then Center aligned text on XL (extra-large) screen and wider.</text>
</code>
  * Output:
</webcode>
