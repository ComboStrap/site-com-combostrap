====== ComboStrap Styling - Class attribute ======


===== About =====
Because all [[docs:component|ComboStrap UI component]] are implemented as [[..:marki|super set HTML element]], you can apply the ''class attributes''.

The ''class'' attribute is used intensively by [[docs:utility:bootstrap|Bootstrap]] to apply the [[https://getbootstrap.com/docs/4.0/utilities/|Bootstrap Utilities]].

''ComboStrap'' already implements most of this utilities through the [[styling#standard attributes|standard attributes]] but in the case that you need something really advanced, you can always take the control over and apply them.
