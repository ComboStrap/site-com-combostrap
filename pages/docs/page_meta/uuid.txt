====== UUID (Universally Unique IDentifier) ======


===== About =====
The ''uuid'' field stores a ''UUID (Universally Unique IDentifier)''. 

An ''uuid'' is a generated page identifier that is unique for the whole internet.


===== Example =====
The ''uuid'' of this page is:

<template>
<code uuid>
$uuid
</code>
</template>
===== Usage =====


  * It is used in the [[docs:analytics:replication|data replication process]] in order to replicate data from the file system to the database
  * It's also useful if you move page files between ComboStrap installation such as:
      * from development to production
      * or if you just split your site web
