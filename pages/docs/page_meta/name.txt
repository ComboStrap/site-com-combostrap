====== ComboStrap Metadata - Page Name ======



===== About =====
The page ''name'' is a metadata that stores the shortest [[..:page_meta:description|description of a page]], generally one or two words.

It can be used in place where long description are not welcome such as the [[docs:navigation:page-explorer|page-explorer]]

This ''name'' will show up:
  * in the [[docs:content:link?preview#preview|link preview as title]].
  * as default name for a [[docs:content:link#internal|internal link]]

===== How to set a name =====

By default, the value of the name is the last part of the [[docs:semantic:path|path]] but you can overwrite it by adding a ''name'' property in the [[..:semantic:frontmatter|frontmatter]]

Example:
<code javascript>
{
    "name":"Page Name"
}
</code>
