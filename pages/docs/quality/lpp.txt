====== Late Publication Protection (LPP) ======


===== About =====
''late publication protection'' is a feature that protects page with a [[docs:semantic:published|published date]] in the future (late) to be disclosed publicly until the [[docs:semantic:published|publication date]] is due.

A visitor should not be anonymous to be able to see or to follow a link to the ''late publication page''.

{{backlinks>.}}

===== Configuration =====


By default, this feature is enable but you can disable it by setting the ''latePublicationProtectionEnable'' [[:configuration|configuration]] to false.

The mode of protection is defined by the [[docs:quality:protection#configuration|protection mode configuration]]
