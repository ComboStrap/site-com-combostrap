====== ComboStrap Metadata - Published Date ======


===== About =====
The ''date_published'' property is a page [[semantic|metadata]] that permits to modify the ''published date'' of an [[..:page_type:article|article]].

By default, the ''published date'' is the ''creation date'' 

<note>
If the ''published date'' is set in the future, the article will be [[docs:quality:lpp|protected]].
</note>

{{backlinks>.}}
===== How to set a published date ? =====



To set a published date in the [[frontmatter|frontmatter]], you add the ''date_published'' property with a [[date|date value]]

Example:
<code json>
{
   "date_published":"2021-03-15"
}
</code>
