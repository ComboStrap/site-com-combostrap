====== ComboStrap Page Metadata - Image ======


===== About =====
''image'' is a [[semantic|page metadata]] that defines the [[docs:content:image|path of an image]] to be used as illustration for a page.

The image is then used on [[docs:social:social|social]] cards such as [[..:social:twitter|twitter]] or [[..:social:facebook|facebook]].

By default, if the ''image'' property is not set, the [[#configuration|first image]] in the page is used.

<note>
[[..:social:twitter#image|twitter]], [[..:social:google#image|google]] and [[..:social:facebook#image|facebook]] have all differents image specifications, therefore when used in this context, the first image conforming to their specification is used.
</note>

===== Definition =====

The image of a page is searched in this order:
  - the ''image'' property in the [[frontmatter|frontmatter]]
  - the ''first image'' of a page if not [[#disableFirstImageAsPageImage|disabled]]
  - the [[docs:app:logo|logo]]

For a [[docs:social:social|social image]] (Twitter, Google, Facebook, ...), if the images found in a step is not conform (low resolution, bad type), the next step is taken.
===== Example =====

Example of a a [[frontmatter|frontmatter]] image:
  * Single: One image
<code json>
{
   "image":":path:beautiful_image.png"
}
</code>
  * Set: The same image in differents resolution and ratio. A set can be used in an [[article#image|article]].
<code json>
{
    "image": [
         ":photos:1x1:photo.jpg",
         ":photos:4x3:photo.jpg",
         ":photos:16x9:photo.jpg"
    ]
}
</code>
  * A [[docs:social:twitter|twitter image]]
<code json>
{
    "image": {
        "twitter":"twitter.jpg"
    }
}
</code>
  * A set of images for [[docs:semantic:article#image|google in a article]]
<code json>
{
    "image": {
        "google":
            [
                ":photos:1x1:photo.jpg",
                ":photos:4x3:photo.jpg",
                ":photos:16x9:photo.jpg"
            ]
    }
}
</code>

<note>
Moving image path with the [[docs:getting_started:4_plugin#move|move plugin]] will update the value in the [[docs:semantic:frontmatter|frontmatter]].
</note>

===== Configuration =====
==== disableFirstImageAsPageImage====
The ''disableFirstImageAsPageImage'' is a [[:configuration|configuration]] that permits to disable the use of the first image of page as the image of a page.
