====== ComboStrap - Metadata ======


===== About =====
The ''metadata manager'' permits to manage and set data for a page. 

It's also known as:
  * web structured data 
  * or semantic web.

The metadatas can be seen:
  * via the [[viewer|Metadata viewer]]
  * or the [[docs:analytics:data|analytics data]]




===== Type =====

There is three groups of metadata:
  * [[#managed]]: the properties that ComboStrap used for its own processing
  * [[#protected]]: the properties that cannot be modified
  * ''unknown'': Unknown properties (They may be written without any further processing)

==== Managed ====

''ComboStrap'' manages and reacts to the following metadata:
  * [[..:page_meta:canonical|canonical]] - an unique page identifier for the web
  * [[docs:page_meta:name|page name]] - a short name for the page
  * [[..:page_meta:title|page title]] - a long name for the page
  * [[..:page_meta:h1|h1]] - the first heading of the page
  * [[..:page_meta:description|description]] - a summary of the page
  * [[docs:semantic:published|date_published]]
  * [[docs:page_meta:image|image]] - the page illustrations
  * [[docs:page_type:page_type|page type]] - the type of page (article, ...)
  * [[published|date_published]] - the publication date
  * [[..:page_meta:country|country]] - the localized country
  * [[..:page_meta:lang|lang]] - the language of the page

==== Protected ====
The below properties are protected and cannot be changed:
  * [[docs:semantic:path|path]] - the path of the page with wiki separator (ie '':namespace:page'')
  * ''date_created'' - the creation date
  * ''date_modified'' - the modification date
  * ''user'', ''creator'', ''contributor''
  * [[docs:cache:page|date_cache_expiration]] - the expiration date of the cache if any
  * [[..:page_meta:uuid|uuid]] - an universal unique identification






===== Usage =====

They are also used in the ''semantic component'' explained in the next section.

==== Internal ====

They allows internally:

  * [[docs:url:redirection|URL redirection processing]]
  * [[..:page_meta:automatic_canonical|Automatic Canonical]]
  * [[docs:url:permalink|Permalink]]
  * [[docs:quality:lpp|Late Publication Protection]]
  * [[docs:templating:template|Templating]]
  * [[docs:cache:page|Cache expiration]]
  * [[docs:analytics:replication|Database replication]]
==== External ====

They allows:
      * a better [[docs:seo:seo|SEO]] with:
          * [[..:page_meta:title|title]]
          * [[..:page_meta:description|description]]
          * [[..:page_meta:canonical|canonical]]
      * a better [[docs:social:social|social experience]] with
           * <icon name="twitter"/> [[..:social:twitter|Twitter card]]
           * <icon name="facebook"/> [[..:social:facebook|Facebook card]]
           * <icon name="google"/> [[..:social:google|Google Rich Result (Organization, Search bar and article card)]]
