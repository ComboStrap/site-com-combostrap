====== ComboStrap - Metadata ======


===== About =====
The ''metadata manager'' permits to manage and set data for a page. 

It's also known as:
  * web structured data 
  * or semantic web.

The metadatas can be seen:
  * via the [[viewer|Metadata viewer]]
  * or the [[docs:analytics:data|analytics data]]


===== Usage =====

They below internal page metadata are used by ''ComboStrap'':
      * [[canonical|canonical]] - an unique page identifier for the web
      * [[description|description]] - a summary of the page
      * [[title|title]] - a long name for the page
      * [[name|name]] - a short name for the page
      * [[type|page Type]] - the type of page (article, ...)
      * [[image|image]] - the image illustrations
      * ''path'' - the path of the page with wiki separator (ie '':namespace:page'')
      * ''date_created'' - the creation date
      * ''date_modified'': the modification date
      * [[published|date_published]] - the publication date
      * [[docs:utility:cache|date_cache_expiration]] - the expiration date of the cache if any
      * [[country|country]] - the localized country
      * [[lang|lang]] - the language of the page

They are also used in the ''semantic component'' explained in the next section.
