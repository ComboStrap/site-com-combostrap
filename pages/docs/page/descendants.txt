====== Descendant Pages ======


===== About =====
''Descendant pages'' are [[docs:layout:page:page|pages]] that have the same or a higher [[docs:page:level|level]] than the current page in the [[docs:page:system|page system]].



The descendants pages can be queried to create illustrative list in [[docs:type:index|home pages]].

===== Query =====

You can query them by using the ''descendants'' table of the [[docs:templating:sql|page sql]].

Example:
  * All descendants [[docs:type:index|index page]] (''is_index = 1'') 
  * in the next [[docs:page:level|level]] (''depth = 1'') than the [[:docs:docs|root home page for the documentation]]
  * ordered by [[docs:page:name|page name]] ascendant
<code sql>
select from descendants where is_index = 1  and depth = 1 order by name asc 
</code>


If you take this query and uses it:
  * in a [[docs:templating:iterator|iterator]] 
  * with a [[howto:template|template]]
<webcode name="Default" frameborder="0">
<code dw>
<iterator>
    <data path=":docs:docs">select from descendants where is_index = 1  and depth = 1 order by path asc limit 10 </data>
    <template>
  * [[$path|$name]] (Level: $level)
    </template>
</iterator>
</code>
you would get the second level [[docs:type:index|index page]] (we have limited to 10 with the ''limit 10'' clause).
</webcode>
