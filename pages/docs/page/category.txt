====== What is the Category of a Page ? ======


===== About =====
In Combostrap, the ''category'' of a page is the [[:docs:page:system|parent page of the page system]].




===== How to print  ? =====
==== the label ====


The parent page name can be printed with the [[:docs:navigation:hierarchical_breadcrumb|typographical breadcrumb]]

Example:
<webcode name="Default" frameborder="0">
  * Markup
<code dw>
This page is in the category: <ibreadcrumb/>
</code>
  * Output:
</webcode>

==== the image ? ====


The image can be printed with the ''descendant'' option of the [[:docs:content:page-image|page image]].

Example:
<webcode name="Default" frameborder="0">
  * Markup
<code dw>
<page-image descendant/>
</code>
  * Output:
</webcode>

===== Usage =====
The category is printed on the [[:docs:content:vignette|vignette]] and can be used to create a [[:docs:layout:page:header_slot|page header]].
