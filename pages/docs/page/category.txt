====== What is the Page Category ? ======


===== About =====
In Combostrap, the ''page category'' is the name of the [[:docs:page:system|parent page of the page system]].


===== How to print the page category ? =====


The page category can be printed with the [[:docs:navigation:hierarchical_breadcrumb|typographical breadcrumb]]

Example:
<webcode name="Default" frameborder="0">
  * Markup
<code dw>
This page is in the category: <ibreadcrumb/>
</code>
  * Output:
</webcode>

===== Usage =====
The category is printed on the [[:docs:content:vignette|vignette]] and can be used to create a [[:docs:layout:page:header_slot|page header]].
