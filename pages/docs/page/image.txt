====== What are the illustrative images for a page? ======

===== About =====
A page has one or more illustrative images.


===== List =====

  * [[:docs:page:featured-image|Featured image]]: the image that is shown just above or below the title
  * [[:docs:social:twitter|Twitter image]]: the image shown in a Twitter card
  * [[:docs:social:facebook|Facebook / Open graph image]]: the image is shown in a Facebook / Open Graph card

===== Rendering =====

These images can be rendered with the [[:docs:content:page-image|page image]] component.

===== Set =====

You can set these images with the [[:docs:metadata:manager|metadata manager]].
