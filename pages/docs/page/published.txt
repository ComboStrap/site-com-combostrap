====== ComboStrap Metadata - Published Date ======


===== About =====
The ''date_published'' property is a page [[..:metadata:metadata|metadata]] that permits to modify the ''published date'' of an [[..:type:article|article]].

By default, the ''published date'' is the ''creation date'' 

<note>
If the ''published date'' is set in the future, the article will be [[docs:quality:lpp|protected]].
</note>


===== How to set a published date ? =====



You can set a published date with the [[docs:metadata:manager|metadata manager]].
