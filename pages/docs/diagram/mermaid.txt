====== ComboStrap - Mermaid diagrams ======


===== About =====
The ''mermaid'' component will render diagrams defined with the [[https://mermaid-js.github.io/mermaid/|mermaid language]]



===== Example =====

==== Flowchart ====

<webcode name="Flow" frameborder="0">
  * The [[https://mermaid-js.github.io/mermaid/#/flowchart|mermaid flow chart definition]]
```dw
<mermaid>
<code mermaid>
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
</code>
</mermaid>
```
  * The output:
</webcode>

==== Sequence ====

<webcode name="Sequence" frameborder="0">
  * The [[https://mermaid-js.github.io/mermaid/#/sequenceDiagram|mermaid sequenceDiagram definition]]
```dw

<code mermaid display="none">

    participant Alice
    participant Bob
    Alice->>John: Hello John, how are you?
    loop Healthcheck

    end
    Note right of John: Rational thoughts <br/>prevail!
    John-->>Alice: Great!

    Bob-->>John: Jolly good!

</mermaid>
```

===== Syntax =====

To render a mermaid diagram, you just need to enclose a [[docs:block:code|code block]] with [[https://mermaid-js.github.io/mermaid/#/|mermaid diagram definition language]] inside a ''mermaid'' block.

```dw
<mermaid>
    <code mermaid display="none">
    ... mermaid diagram definition
    </code>
</railroad>
```

<note>
''display="none"'' is added to show you how to not print a [[docs:block:code|code block]] but this is optional.
</note>
