====== ComboStrap Diagram - Gant ======

===== About =====
The ''gant'' component is a [[diagram|diagram component]] that permits to create ''gant diagram''


===== Example =====


<webcode name="Gant" frameborder="0">
  * The markup
```dw
<gant>
gantt
    title A Gantt Diagram
    dateFormat  YYYY-MM-DD
    section Section
    A task           :a1, 2014-01-01, 30d
    Another task     :after a1  , 20d
    section Another
    Task in sec      :2014-01-12  , 12d
    another task      : 24d
</gant>
```
  * The output:
</webcode>

===== Syntax =====

The syntax is based on the  [[https://mermaid-js.github.io/mermaid/#/gantt|mermaid Gant definition]]

<code dw>
<gant>
   ... Mermaid Gant definition ...
</gant>
</code>
