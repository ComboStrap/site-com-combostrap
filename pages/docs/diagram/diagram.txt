====== ComboStrap Diagram ======


===== About =====
''ComboStrap'' is able to create and render diagrams.

If your diagrams are in a ''svg'' format, you can just render them with the [[docs:content:svg|svg component]].

Otherwise, you can use the following components to create and render diagram.
  * [[class]]
  * [[erd]]
  * [[flowchart]]
  * [[gant]]
  * [[pie]]
  * [[railroad]]
  * [[sequence]]
  * [[state]]
  * [[user-journey]]



Example with a [[sequence|sequence Diagram]].

<mermaid width="500px">
<code mermaid display="none">
sequenceDiagram
    participant Editor
    participant ComboStrap
