====== ComboStrap Page Metadata - Type of Page ======


===== About =====
''type'' is a [[docs:semantic:semantic|metadata]] that define the type of a page. 

This information is used by:
  * the [[docs:social:facebook|Facebook Social Graph]].
  * and [[docs:social:google|Google Rich Result]]
to set the properties that they required when crawling your website.


===== How to set the value =====
You can set the value in the [[docs:semantic:frontmatter|frontmatter]]

<code json>
{
    "type":"article"
}
</code>

If not set, the default value is:
  * [[docs:page_type:website|website]] for the [[docs:page_type:home#root|root home page]]
  * [[docs:page_type:home|home]] for other [[docs:page_type:home|home page]]
  * and the value of the [[#configuration|default configuration]] for the others ([[docs:page_type:article|article]] at installation time)

===== Configuration =====

==== defaultPageType ====
The ''defaultPageType'' [[:configuration|configuration]] permits to set the default ''type'' value of a page except for the home page that has a default value of [[docs:page_type:website|website]].

By default, the value is [[docs:page_type:article|article]].
