====== ComboStrap URL Management - Path Redirect ======


===== About =====
An ''path redirect'' is a transparent [[redirection|redirection]] that sets the [[docs:metadata:path|path]] to a new value.

It can happen in the following case:
  * If the URL matches a [[#canonical|canonial path]], the path will be rewritten
  * If the URL matches a [[docs:metadata:automatic_canonical|automatic canonial path]], the path will be rewritten
  * If the [[page_rules|page rules redirection]] leads to an existent page.
  * If the the [[bestendpagename|best end page name scoring]] is greater or equal than the [[bestendpagename#configuration|Minimal Score For Id Redirect configuration]]

==== Canonical Processing ====

  * When a user is asking for a [[404|missing page]] with the [[docs:metadata:path|path]] value ''namespace:name'' via one of this URL:
      * https://example.com/doku.php?id=namespace:name
      * or https://example.com/namespace/name (if [[doku>rewrite|dokuwiki URL rewriting]] was set)
  * If the page has a [[docs:metadata:canonical|canonical value]] of ''othernamespace:name'', the ''path'' will be rewritten (set) from:
      * ''namespace:name''
      * to ''othernamespace:name''


===== Message =====
A ''path redirect'' is transparent and shows no message.
