====== ComboStrap URL Management - Transparent Redirect ======


===== About =====
A ''transparent redirect'' is a [[redirection|redirection]] that shows the content of another page transparently (A visitor or a search engine bot will not see the redirection).

It can happen in the following case:
  * If the [[docs:metadata:path|path]] in the URL matches the [[docs:metadata:canonical|canonical path]] of a page
  * If the [[docs:metadata:path|path]] in the URL matches a [[docs:metadata:alias#synonym|synonym alias]] of a page
  * If the [[docs:metadata:path|path]] in the URL matches the [[docs:metadata:uuid|uuid]] of a page
  * If the [[docs:metadata:path|path]] in the URL matches a [[docs:metadata:automatic_canonical|automatic canonical path]]

===== Processing explanation =====

  * When a user is asking for a [[404|missing page]] with the [[docs:metadata:path|path]] value ''namespace:name'' via one of this URL:
      * https://example.com/doku.php?id=namespace:name
      * or https://example.com/namespace/name (if [[doku>rewrite|dokuwiki URL rewriting]] was set)
  * If a page is found with a [[docs:metadata:canonical|canonical]] or a [[docs:metadata:alias|synonym alias]] that has the value ''othernamespace:name''
  * This page content is shown


===== Message =====
A ''transparent redirect'' is silent and shows no message.
