====== ComboStrap UI - Template ======

===== About =====
''template'' are text with [[#variable|variables]] that are replaced at runtime.

By default, the variable value are from the requested page.

The variable value are from the selected pages when used inside a dynamic component. ie:
  * in the [[docs:navigation:page-explorer|page explorer component]] to print information of the page of the file system
  * in the [[docs:templating:iterator|iterator component]] to print information of selected pages.


===== Example: Printing information about the actual page =====

The ''template text'' will print a [[docs:lists:list|bullet list]] with the [[docs:page:title|title]] and [[docs:page:path|path]] of the actual page.
<webcode name="list">
<code dw>
  * Title: $title
  * Path: $path 
</code>
</webcode>





===== Syntax=====


A ''template text '' can contains any [[docs:component|component]] and zero or more [[variable|variable]] (that supports text transformation through the use of a [[docs:templating:pipeline|pipeline]])
<code dw>
A template text with a $variable
</code>
