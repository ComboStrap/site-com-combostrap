====== ComboStrap UI - Template ======


===== About =====
''template'' are text that may contains [[#variable|variables]] that will be replaced at runtime.

They are supported:
  * in its own [[#syntax|template component]] (to print information of the requested page)
  * in the [[docs:navigation:page-explorer|page explorer component]] (to print information of the page on the file system)
  * in the [[docs:templating:iterator|iterator component]] (to print information of selected pages)

===== Example =====

==== Printing information about the actual page ====

The below ''template text'' inside a ''template component'' will print a [[docs:lists:list|bullet list]] with the [[docs:page:title|title]] and [[docs:page:path|path]] of the actual page.
<webcode name="list">
<code dw>
<template>
  * Title: $title
  * Path: $path 
</template>
</code>
</webcode>


==== With pipeline====


Example of ''template text'' that you could used in a [[docs:navigation:page-explorer|page-explorer navigation panel]].

It will create:
  * a [[docs:content:link|link]] 
  * with a [[docs:templating:pipeline|pipeline]] 
  * that separate the [[docs:page:h1|h1]] with the minus ''-'' 
  * and used the second part as link description

<webcode name="pipeline" frameborder="0">
<code dw>
<template>
The [[$path|<pipeline>"$h1" | cut("-","2-") | trim()</pipeline> page]].
</template>
</code>
<hr/>
**Result:**
</webcode>


===== Syntax=====

==== Template Text ====

A ''template text '' can contains any [[docs:component|component]] and zero or more [[#variable|variable]] (that supports text transformation through the use of a [[docs:templating:pipeline|pipeline]])
<code dw>
A template text with a $variable
</code>

==== Template component ====

A ''template'' component contains a ''template text''
<code dw>
<template>
The template text with $variable
</template>
</code>

If this is a standalone template component, the variable are replaced with the values of the actual requested page, otherwise they are replaced by the value of the requested data (ie in a [[docs:templating:iterator|iterator]])

==== Variable ====

A variable is the name of an [[variable|attribute]] prefixed with the ''$'' dollar character.

Example:
  * for the [[docs:page:path|path]] attribute
  * the variable would be ''$path''
