====== ComboStrap - Iterator ======

===== About =====
''iterator'' permits to loop and applies a [[docs:templating:template|template]] repeatably in order to create a list of pages.

You may choose to
  * [[docs:layout:component:masonry|masonry]]
  * carrousel
  * [[docs:lists:list|simple list]]
  * [[docs:lists:content-list|content list]]
  * [[docs:layout:component:grid|grid]]


===== Example =====
==== Masonry ====

<webcode name="Default" frameborder="0">
<code dw>
<iterator>
   <data>select date_modified, path, h1, description, name order by date_modified desc limit 8</data>
   <body>
          <heading d2 text-align="center" spacing="mb-5"> The last 8 Changes </heading>
          <masonry>
              <template>
                  <card>
                      === $h1 ===
                      $description
              
                      <btn secondary>[[$path|$name]]</btn>
                  </card>
              </template>
          </masonry>
   </body>
</iterator>
</code>
</webcode>


===== Syntax =====
<code dw>
<iterator>
   <data>select attribute1, attribute2 where attribute3 = 'xxx'</data>
   <body>$attribute1 ... $attribute2</body>
</iterator>
</code>
where:
  * ''data'' defined a sql statement:
      * that may contains any [[docs:templating:template#variable|template variable]]
      * without ''from'' clause
      * with an optional ''where'' clause
      * with an optional ''order by'' clause
      * with an optional ''limit'' clause
  * ''body'' contains a [[docs:templating:template|template]] that will be repeated for each data row.


<note>
If your iteration is time-based, you may need to set a cache expiration on your page with the [[docs:utility:cache|cache]] component
</note>
