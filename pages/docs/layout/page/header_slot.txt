====== ComboStrap - Page Header Slot ======


===== About =====
The ''page header slot'' is a [[docs:layout:page:slot|slot]] that contains the ''header'' of the [[page|page]].

The ''page header'' is rendered at the top of the page and contains generally a [[docs:menubar:menubar|menu bar]]. 




<note>
  * The ''how-to'' getting started guide explains how to create a header navigational bar: [[docs:getting_started:6_menu]].
  * It defines a header for the page. If you want to define a header for a component, see the [[docs:utility:header|header component]]
</note>


===== Configuration =====
You can configure the name of the ''header slot'' with the ''tpl»strap»headerslot'' [[docs:app:configuration|configuration]]
