====== Strap Template - Sidekick Slot ======

===== About =====
The ''sidekick slot'' is a [[docs:layout:page:slot|slot]] that is shown at the right side of the  [[docs:layout:page:main|main content]]..



===== How it works =====
This is a standard [[docs:layout:page:slot|slot feature]]

  * Create a page called ''slot_sidekick'' (The [[#name|default configuration value]])
  * And you will see its content appears at the right side of the main content.


===== Configuration =====
==== Name ====

You can configure the name of the ''sidekickslot'' with the ''tpl»strap»sidekickslot'' [[docs:app:configuration|configuration]]
