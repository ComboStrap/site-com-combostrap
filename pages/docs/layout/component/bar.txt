====== ComboStrap UI - Bar component ======

===== About =====
A ''bar'' is an [[docs:component|ui component]] that divides a page in horizontal region.

 through the use of [[docs:styling:background|background]] (and therefore without any [[docs:styling:spacing|margin]]).

''bar'' are used:
  * in the [[:slot_footer|page footer]] 
  * or in [[:docs:layout:page:landing|landing page]].




===== Syntax =====

<code dw>
<bar size="md|sm|lg|xl">
   <background></background>
   <backgrounds>
       <background></background>
       ...
       <background></background>
   </backgrounds>
   <separator></separator>
   My horizontal bar
</bar>
</code>
where:
  * ''size'' defines the space around the text (''padding'') and is one of:
      * ''sm'' : ''small''
      * ''md'' : ''medium'' (default)
      * ''lg'' : ''large''
      * ''xl'' : ''extra-large''
  * ''background'' is the optional [[docs:styling:background|background definition]]

<note>
This is also an [[docs:marki|super HTML component]] and accepts therefore all [[docs:styling:styling|styling attributes]]
</note>
