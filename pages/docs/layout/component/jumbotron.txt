====== ComboStrap UI - Jumbotron ======


===== About =====
A ''jumbotron'' is a simple [[docs:block:block|box component]] that emphase prominently its content.

<note>
For a hero unit style content, you can also use a [[docs:layout:component:slide|slide]].
</note>


===== Articles Related =====
{{backlinks>.}}

===== Syntax =====
<code dw>
<jumbotron>
=== Title ===
Content
</jumbotron>
</code>
