====== ComboStrap UI - Tooltip ======


===== About =====
A ''Tooltip'' displays informative text when users hover over, focus on, or tap an element.

===== Articles Related =====
{{backlinks>.}}

===== Syntax =====
A ''tooltip'' is a [[docs:component|UI component]] that adds a tooltip to its container generally a button or a link.
<code dw>
<tooltip position="top">
The tooltip content
</tooltip>
</code>
where:
  * ''position'' is one of ''top'' (Default),''right'',''bottom'', ''left''

===== Example =====

==== A link with a custom tooltip ====
Because a link should be written on one line, you need to inline the tooltip markup.
<webcode name="Default">
<code dw>
[[#|<tooltip position="right">A tooltip</tooltip>A link with a tooltip on the right]]
</code>
</webcode>

==== A link with the default tooltip ====

A [[docs:content:link|link]] can get a tooltip created from the [[docs:semantic:title|title]] and the [[docs:semantic:description|description]] via the ''tooltip'' attribute.
<webcode name="Default">
<code dw>
[[?tooltip|A link with a tooltip]]
</code>
</webcode>
==== Button ====

<webcode name="Button">
<code dw>
<btn>
<tooltip position="top">
=== Tooltip Heading ===
A tooltip text
</tooltip>
A button with a tooltip on the top
</btn>
</code>
</webcode>
