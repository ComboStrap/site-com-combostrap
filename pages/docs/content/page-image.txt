====== Page-image Component: Renders the illustrative image of your page ======


===== About =====
''page-image'' is a [[content|content component]] that chooses and renders one of the [[docs:page:image|page images]] (illustration, thumbnail) defined for a page.






**Example:** This example shows the [[docs:page:image|page image]] of this page. We have set it to the image of the surfer below.

<webcode name="Default" frameborder="0">
<code dw>
<page-image/>
</code>
</webcode>





===== Syntax =====

<code dw>
<page-image 
   type="type"
   default="meta|first|descendant|vignette|logo"
   path="path"
/>
</code>
where 
  * ''path'' is the [[docs:page:path|page path]] to show the image for. Default to the actual page path or the [[:docs:templating:iterator|path given by the iterator]] 
  * [[:docs:styling:type|type]] is one of:
      * ''meta'' is the [[:docs:content:page-image|page image meta]] (default)
      * ''first'' is the first image of the page
      * ''vignette'' is the [[vignette|vignette]]
      * ''descendant'' is the first image found in the [[:docs:page:system|descendant page]] (parent and grand parent)
      * ''logo'' is the [[:docs:app:logo|logo]]
  * ''default'' determines the default image type chosen in order of precedence (default to ''meta|first|descendant|vignette|logo'')


<note>
  * All [[docs:content:image|image attributes]] such ''width'' / ''height'' / ''ratio'' and ''color'' and more can be applied.
  * For an [[docs:content:icon|icon]], the default color is the [[docs:app:color|primary color]] if set
</note>


 
===== Image selection =====

When the ''ratio'' is set, the algorithm will:
  * select the image in the set of [[docs:page:image|page image]] with the most equivalent ratio
  * [[docs:content:image#cropping|crop]] the image to fit the ratio


===== Support =====

==== Icon Support ====

When the image is an [[docs:content:icon|icon]], the icon is zoomed out 3 times and centered.

<webcode name="Default" frameborder="0">
<code dw>
<page-image path=":docs:content:icon" ratio="16:9" color="pink" width="198" border-color="pink"/>
</code>
</webcode>


==== Card Support ====


A page-image is supported as illustrative image for a [[docs:block:card|card]]

Example a card with data of this page with [[:docs:templating:variable|variable]]
<webcode name="Default" frameborder="0">
<code dw>
<card clickable width="300">
    <page-image ratio="16:9"/>
    === $h1 ===
    $description [[$path| ]]
</card>
</code>
</webcode>
