====== ComboStrap UI - Cite ======


===== About =====
The ''cite'' element will give credit to a portion of text. It can go anywhere but you put it mostly in a [[docs:block:blockquote|blockquote]].


===== Example =====

==== Basic ====

  * Markup
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<cite>The credits goes to</cite>
</code>
  * Output:
</webcode>

==== With style ====
You can [[docs:styling:styling|style]] them. For instance, with a ''blue'' [[docs:styling:color|color]]
  * Markup
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<cite color="blue">The credits goes to</cite>
</code>
  * Output:
</webcode>

==== With Blockquote ====
With a [[docs:block:blockquote|blockquote]]
  * Markup
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<blockquote>
    It's not the Destination, It's the journey.
    <cite>Ralph Waldo Emerson</cite>
</blockquote>
</code>
  * Output:
</webcode>
