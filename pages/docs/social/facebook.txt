====== ComboStrap Page Metadata - Facebook Card ======


===== About =====
Sharing a link will show on every facebook post a card thanks to the facebook social graph metadata added to each ComboStrap page.



<badge>Since [[release:release#version_112|version 1.12]]</badge>

===== Articles Related =====
{{backlinks>.}}
===== Preview =====


You can check the generated card before posting with the [[https://developers.facebook.com/docs/sharing/webmasters#testing|sharing debugger]].

</box>

===== List of metadata =====

The following metadata's are used in the card:
  * for the ''og:title'': [[..:page:title|title]] if present otherwise [[..:page:h1|h1]], otherwise the page id.
  * for the ''og:description'': [[..:page:description|description]]
  * for the ''og:image'': the [[..:page:image|image]]. The image must be at minimum 200px by 200px (The card rendering uses 524x274) More see the [[https://developers.facebook.com/docs/sharing/webmasters/images|Facebook Image documentation]]
  * for the ''og:type'': the [[..:semantic:type|page type]]
  * for the ''og:locale'': the [[..:locale:locale|locale]] - if the locale cannot be derived, we set the default Facebook value (''en_US'').

<note>
If the page is of the [[..:type:article|article type]], we add also the [[..:type:article|article metadata]]
</note>
===== Configuration =====

You can set the following [[docs:app:configuration|configuration]]:

^ Name ^ Description ^ Default value ^
| ''facebookDefautImage'' | the default image if no [[..:page:image|page image]] was found | :logo-facebook.png |
