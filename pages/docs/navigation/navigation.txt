====== ComboStrap UI - Navigation ======


===== About =====
''navigational components'' helps the user to navigate and discover your website.


===== List =====

[[docs:component|ComboStrap UI]] get the following component in its library.

==== Block Component ====


  * [[docs:navigation:breadcrumb]] : A list of pages shown in a [[docs:navigation:hierarchical_breadcrumb|hierarchical]] or [[docs:navigation:historical_breadcrumb|historical]]  way.
  * [[docs:menubar:menubar]] : The top navigation bar
  * [[docs:theme:slot:toc]] : The table of content
  * [[docs:navigation:page-explorer]] : A page navigation component that shows a list or a tree of pages
  * [[docs:layout:component:tabs]] : A tab when activated that shows its related [[docs:layout:component:panel|panel]]
  * [[docs:layout:component:accordion]] : A list of rows that when activated shows its related [[docs:layout:component:panel|panel]]

==== Text Component ====

  * [[docs:menubar:search|Search]] - A search form
  * [[docs:content:button|Button]] - A button
  * [[docs:content:link|Link]] - A link
