====== What are the Navigational components in Combostrap? ======


===== About =====
''navigational components'' helps the user to navigate and discover your website.



===== Label =====
All navigation components use by default the [[:docs:page:name|page name]] as label.

===== List =====

[[docs:component|ComboStrap UI]] has the following navigational components in its library.

==== Block Component ====


  * [[docs:navigation:breadcrumb|Breadcrumb]] : A list of pages shown in a [[docs:navigation:hierarchical_breadcrumb|hierarchical]] or [[docs:navigation:historical_breadcrumb|historical]]  way.
  * [[docs:menubar:menubar|Menubar]] : The top navigation bar
  * [[docs:theme:element:toc|Toc]] : The table of content
  * [[docs:navigation:page-explorer|Page explorer]] : A page navigation component that shows a list or a tree of pages
  * [[docs:layout:component:tabs|Tabs]] : A tab when activated that shows its related [[docs:layout:component:panel|panel]]
  * [[docs:layout:component:accordion|Accordion]] : A list of rows that when activated shows its related [[docs:layout:component:panel|panel]]

==== Text Component ====

  * [[docs:menubar:search|Search]] - A search form
  * [[docs:content:button|Button]] - A button
  * [[docs:content:link|Link]] - A link
