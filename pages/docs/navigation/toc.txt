====== ComboStrap UI - Table of Content (TOC) ======

===== About =====
The ''toc (table of content)'' gives to the reader an overview of your page.

The ''toc'' is composed of the [[docs:navigation:outline|outline headings]].


===== Features =====

If you uses the [[docs:utility:renderer|combo renderer]], the TOC is printed below the [[docs:page:h1|first heading]].







===== Configuration =====
The TOC component supports: 
  * all [[doku>toc#configuration|native configuration]] such as:
      *  [[doku>config:tocminheads]]
      * [[doku>config:maxtoclevel]]
      * [[doku>config:toptoclevel]]
  * the ''No toc'' tag to disable the [[doku>toc]] rendering
