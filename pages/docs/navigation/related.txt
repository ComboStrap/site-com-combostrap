====== ComboStrap UI - Related ======


===== About =====
''related'' is a [[docs:component|ComboStrap UI component]] that shows the [[backlink|backlinks]] of a page ordered by their importance.


The more backlinks has a page, the more important it is.



===== Syntax =====

<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<related/>
</code>
</webcode>

===== Configuration =====
==== Maximum number of backlinks ====
The max numbers of backlinks can be set. Default value: ''10''

==== Link Content ====
The link content follows the [[doku>config:useheading]] configuration.

==== Extra Pattern ====

An extra pattern can also be set if you come for instance from the [[doku>plugin:backlinks|backlinks plugin]]. Default value: ''%%{{backlinks>.}}%%''
