====== ComboStrap UI - Table ======


===== About =====
''Table'' is a component that shows data in a [[lists|list]] of record.




The table is responsive and adapts to all screen sizes.

<note>
If you want to use a table for layout purpose, you should use a [[docs:layout:component:grid|grid]]
</note>

==== Enable renderer ====
