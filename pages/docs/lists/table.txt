====== ComboStrap UI - Table ======


===== About =====
''Table'' is a component that shows data in a [[lists|list]] of record.




The table is responsive and adapts to all [[:docs:styling:breakpoint|screen sizes]] but does not wrap. If you want to use a table for layout purpose, you should use a [[docs:layout:component:grid|grid]]






===== Example =====


<webcode name="Tables" >
  * Input
<code dw>
^ Heading 1      ^ Heading 2       ^ Heading 3          ^
| Row 1 Col 1    | Row 1 Col 2     | Row 1 Col 3        |
| Row 2 Col 1    | some colspan (note the double pipe) ||
| Row 3 Col 1    | Row 3 Col 2     | Row 3 Col 3        |
</code>
  * Output:
</webcode>

===== Configuration =====
==== Enable renderer ====
The table rendering is a feature of the [[docs:utility:renderer|renderer]] and should be then selected. See [[docs:utility:renderer#configuration|renderer configuration]].
