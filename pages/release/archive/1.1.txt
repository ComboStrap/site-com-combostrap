====== ComboStrap - Release 1.1 ======

===== About =====
[[..:release|Release]] 1.1 is the second ComboStrap released from 2020-07-10

===== Changes List =====

  * The [[docs:semantic:viewer|metadata viewer]] shows now all metadatas and can apply a filter
  * The [[docs:semantic:title|web page title]] is now managed.
  * The [[docs:block:math|math component]] is no more template dependent.
  * The [[docs:utility:disqus|disqus component]] was finalized
  * The [[docs:content:badge|badge component]] was added
