====== Deprecated Combo Renderer (Rplus) ======

===== About =====
The ''combo renderer'' has been deprecated because the features that it was implementing are now integrated into the Combo plugin.


The features were:

  * Make the [[docs:lists:table|tables responsive]]
  * Enable [[docs:ads:ads|ads inside the content]]


===== You should disable the Combo Renderer? =====

You need to unset the [[doku>config:renderer_xhtml|renderer_xhtml]] [[docs:app:configuration|configuration]]
