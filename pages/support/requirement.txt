====== ComboStrap - Platform Requirements ======


===== About =====
This articles lists the installation prerequisites of [[:combostrap|ComboStrap]].

===== List =====
==== Php ====

ComboStrap follows the [[doku>requirements|Dokuwiki Requirement]] ([[https://github.com/splitbrain/dokuwiki/blob/master/.github/workflows/testLinux.yml#L13|Build Ref]], [[https://github.com/splitbrain/dokuwiki/blob/master/composer.json#L8|Composer Ref]]) and is therefore tested against ''php'' version 7.2, 7.3 to 7.4.

<note warning>
  * php 8.0 is not [[https://github.com/splitbrain/dokuwiki/issues/3399|working yet]]
  * At php, the version 7.2 is already in a [[https://www.php.net/eol.php|End Of Life state]]. You can check the [[https://www.php.net/supported-versions.php|timeline support for the php version]]
</note>
==== Module / Extension ====

ComboStrap uses [[https://www.php.net/manual/en/dom.installation.php|Php DOM]] for [[docs:content:svg|svg]] and [[docs:content:icon|icon]] manipulation.



ComboStrap is dependent on the following extensions
  * ''pdo_sqlite''
  * ''mbstring''
  * ''openssl''

They should be available on standard php installation, if this is not the case, you can install/configure them

Example:
  * on Windows in the ''php.ini'' file
<code ini php.ini>
[PHP]
extension=php_pdo_sqlite.dll
extension=php_mbstring.dll
extension=php_openssl.dll
</code>
  * on Linux
<code bash>
# ubuntu, ...
sudo apt-get install php-xml php-pdo-sqlite php-mbstring php-openssl
# centos, redhat
sudo yum install php-xml php-pdo-sqlite php-mbstring php-openssl
</code>
