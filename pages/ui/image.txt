====== ComboStrap UI - Image ======

===== About =====
''image'' is a [[block|block UI Component]] that permits to show [[raster|raster]] or [[svg|svg]] images in your pages (We should say media because video are also supported)


<note>
All images are [[:lazy|lazy loaded]] by default to enhance [[:performance|performance]]
</note>
===== Articles Related =====
{{backlinks>.}}

===== Syntax =====

ComboStrap uses the same syntax than [[doku>wiki:syntax#media_files|Dokuwiki]]

<code dw>
{{[ ]path?WxH[ ]}}
</code>
where:
  * ''path'' is:
      * for an internal image: the internal path of the dokuwiki
      * for an external image: the URL.
  * ''W'' is the width (0 = no width defined)
  * ''H'' is the height (Optional)
  * ''[ ]'' are optional spaces that indicates the alignement of the image (left or right)



===== Example =====

==== Resize to given width ====
  * Internal image
<code dw>
{{wiki:dokuwiki-128.png?200}}
</code>
  * External Image
<code dw>
{{https://secure.php.net/images/php.gif?200}}
</code>
==== Resize to given width and height ====
  * Internal image
<code dw>
{{wiki:dokuwiki-128.png?200x50}}
</code>
  * External Image
<code dw>
{{https://secure.php.net/images/php.gif?200x50}}
</code>

==== Elevation ====

You can style even further your image with [[styling:styling|styling element]]:
  * with a [[styling:shadow|shadow elevation through styling]].
  * with a [[styling:hover|hover float effect]]

For instance, to elevate this photography with a surfer, you could use this syntax.

<webcode name="Default" frameborder=0>
<code dw>
<box elevation="high" align="center" width="fit-content" onHover="float">
  {{:ui:stock_image_surfer_in_the_see.png?600|}}
</box>
</code>
</webcode>
