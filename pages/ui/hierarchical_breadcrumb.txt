====== ComboStrap UI - Hierarchical Breadcrumb ======


===== About =====
A ''Hierarchical Breadcrumb'' is a [[breadcrumb|breadcrumb]] that shows where the page is located in the web site tree. 

It's a [[navigation|navigational component]] that reflects the hierarchy of the URL and gives the reader the possibility to navigate to the parent (ie the start page).

===== Configuration =====
==== Enable ====
The ''hierarchical breadcrumb'' is shown when:
  * the [[renderer|renderer is the combo renderer]]. 
  * the [[doku>conf:youarehere|youarehere]] [[:configuration|configuration]] is set to ''on''
